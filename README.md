# Wahlumfragen Pferderennen

Visualisierung aktueller Bundestags-Wahlumfragen im Stil eines Pferderennens.

**[Live-Demo](https://yeah69.github.io/DawumHorseRace/)**

## Datenquelle

Die Umfragedaten stammen von [dawum.de](https://dawum.de/Bundestag/) und stehen unter der ODC-ODbL-Lizenz.
